🚀 QUICK START:

1. Install dependencies:
   npm install expo-linear-gradient @expo/vector-icons

2. Create route files (see MOBILE_SETUP.md for details):
   app/(auth)/login.tsx
   app/(auth)/signup.tsx
   app/entry-selection.tsx
   app/new-visitor-form.tsx

3. Import and use screens:
   import { Login, Signup, EntrySelection, NewVisitorForm } from '@/screens';

4. Start your dev server:
   npm start
